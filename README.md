# Movie Project

Bu projede kullanıcının oturum açarak bazı kategorideki filmleri görebildiği, arama yapabildiği ve ayarlardan tema değiştirebildiği, bilgilerini güncelleyebildiği ve çıkış yapabildiği bir uygulama amaçlanmıştır.

Kullanıcı, bilgilerini girerek uygulamaya kayıt olur. Sign Up butonuna basıldığında kullanıcı bilgileri json-server paketi ile bilgisayarda bir dosyaya yazılır.
